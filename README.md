## 이메일 템플릿 퍼블리싱 간 참고사항

  - 현존하는 다양한 브라우저들의 호환성을 고려해 table,tbody,tr,td만 사용하여 코드를 구성합니다.
   그리고 엑셀처럼 행과 열의 구조를 이해해야 하며 각 table 마다 태그 안에 직접 style을 넣어 여백 등을
   조정 해야합니다.
